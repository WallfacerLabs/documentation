---
title: "Transactions"
description: "Ready-to-sign transaction payloads for deposits, redemptions, and reward claims."
---

Generates ready-to-sign transaction calldata for any supported vault action. Pass the output directly to your wallet provider — no ABI encoding required.

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | [/v2/transactions/context/\{userAddress\}/\{network\}/\{vaultId\}](/api-reference/transactions/get-transaction-context) | Balances and limits before building a transaction |
| `GET` | [/v2/transactions/\{action\}/\{userAddress\}/\{network\}/\{vaultId\}](/api-reference/transactions/get-transaction-payload) | Calldata for deposit, redeem, or claim |
| `GET` | [/v2/transactions/rewards/context/\{userAddress\}](/api-reference/transactions/get-rewards-context) | Claimable rewards for a wallet |
| `GET` | [/v2/transactions/rewards/claim/\{userAddress\}](/api-reference/transactions/claim-rewards) | Calldata to claim pending rewards |
| `GET` | [/v2/transactions/suffix/\{userAddress\}/\{vaultId\}](/api-reference/transactions/get-transaction-suffix) | Onchain referral builder code to append to calldata |
