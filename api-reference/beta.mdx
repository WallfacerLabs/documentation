<Warning>
  Beta endpoints are functional and tested but not yet part of the stable API. Interfaces may change before general availability. Not recommended for production use without monitoring.
</Warning>

Beta features are accessible using your existing API key. The endpoints below cover two areas currently in testing:

**The Deep** — AMM protocol support. Deposit, withdraw, and track positions for Deep liquidity vaults across EVM networks.

**Fixed Term (Pendle)** — Fixed-term yield markets. Retrieve Pendle market details and generate swap-in/swap-out transactions for fixed-rate positions.

If you're building on a Beta endpoint and want to stay informed of changes, [contact us](https://vaultsfyi.fillout.com/t/tLA9t6HyAFus).
