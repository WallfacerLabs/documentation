<Warning>
  Alpha endpoints are experimental and subject to breaking changes at any time. They are not included in the standard API changelog. Use them to explore upcoming features, not in production flows that require stability.
</Warning>

Alpha features are accessible using your existing API key. The endpoints below cover four areas in early development:

**Borrow (Aave & Morpho)** — Position data and transaction payloads for borrowing against Aave and Morpho markets.

**Solana (SVM)** — Vault listings, positions, and transaction support for Solana-based yield vaults.

**Cross-chain transactions** — Intent-based quote, execution, and status tracking for cross-chain position changes.

**Holders** — Vault holder data with LP token balance conversions.

If you're interested in any of these areas and want early access or to provide feedback, [contact us](https://vaultsfyi.fillout.com/t/tLA9t6HyAFus).
