---
title: "Detailed Vaults"
description: "Comprehensive vault data including APY, TVL, risk scores, fees, holder data, and protocol details."
---

The Detailed Vaults endpoints return the full vault object — everything you need to display a vault page, compare vaults, or build a yield aggregator. For a lightweight index of vault addresses and assets only, use [General → /v2/vaults](/api/general).

---

## GET /v2/detailed-vaults

Returns a filterable, paginated list of vaults with complete performance metrics.

```bash
GET https://api.vaults.fyi/v2/detailed-vaults
```

### Query parameters

**Pagination**

<ParamField query="page" type="integer" default="0">Page number (zero-indexed).</ParamField>
<ParamField query="perPage" type="integer" default="50">Items per page.</ParamField>

**Asset & Protocol filters**

<ParamField query="allowedAssets" type="array">
  Assets to include, by symbol (e.g. `USDC`, `WETH`). When set, `disallowedAssets` is ignored.
</ParamField>
<ParamField query="disallowedAssets" type="array">
  Assets to exclude, by symbol. Ignored when `allowedAssets` is set.
</ParamField>
<ParamField query="allowedProtocols" type="array">
  Protocol names to include. When set, `disallowedProtocols` is ignored.
</ParamField>
<ParamField query="disallowedProtocols" type="array">
  Protocol names to exclude. Ignored when `allowedProtocols` is set.
</ParamField>

**Network filters**

<ParamField query="allowedNetworks" type="array" default='["base","mainnet","arbitrum","optimism"]'>
  Networks to include (name or CAIP-2). When set, `disallowedNetworks` is ignored.
</ParamField>
<ParamField query="disallowedNetworks" type="array">
  Networks to exclude. Ignored when `allowedNetworks` is set.
</ParamField>

**TVL & APY filters**

<ParamField query="minTvl" type="integer" default="100000">Minimum vault TVL in USD.</ParamField>
<ParamField query="maxTvl" type="integer">Maximum vault TVL in USD.</ParamField>
<ParamField query="minApy" type="number">Minimum APY as a decimal (e.g. `0.05` = 5%).</ParamField>
<ParamField query="maxApy" type="number">Maximum APY as a decimal.</ParamField>

**Score & quality filters**

<ParamField query="minVaultScore" type="number">Minimum vaults.fyi reputation score (0–100).</ParamField>
<ParamField query="onlyTransactional" type="boolean">Only return vaults that support deposit/redeem transactions.</ParamField>
<ParamField query="onlyAppFeatured" type="boolean">Only return vaults featured on app.vaults.fyi.</ParamField>
<ParamField query="allowVaultsWithWarnings" type="boolean">Whether to include vaults that carry active warnings. Defaults to `true`.</ParamField>
<ParamField query="allowCorrupted" type="boolean" default="false">Include vaults with data integrity issues.</ParamField>

**Taxonomy filters**

<ParamField query="tags" type="array">Filter by vault tags (e.g. `restaking`, `rwa`, `delta-neutral`).</ParamField>
<ParamField query="curators" type="array">Filter by curator name.</ParamField>

**Sorting**

<ParamField query="sortBy" type="string">Field to sort by: `tvl`, `apy1day`, `apy7day`, `apy30day`.</ParamField>
<ParamField query="sortOrder" type="string" default="asc">Sort direction: `asc` or `desc`.</ParamField>

### Example

```bash
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.vaults.fyi/v2/detailed-vaults?allowedAssets=USDC&allowedNetworks=mainnet,base&minTvl=1000000&sortBy=apy7day&sortOrder=desc&perPage=5"
```

## GET /v2/detailed-vaults/{network}/{vaultAddress}

Returns the full vault object for a single vault. Same response shape as a single item from the list endpoint above.

```bash
GET https://api.vaults.fyi/v2/detailed-vaults/{network}/{vaultAddress}
```

### Path parameters

<ParamField path="network" type="string" required>
  Network name (e.g. `mainnet`, `base`) or CAIP-2 identifier.
</ParamField>
<ParamField path="vaultAddress" type="string" required>
  Vault contract address.
</ParamField>

### Example

```bash
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.vaults.fyi/v2/detailed-vaults/mainnet/0x9D39A5DE30e57443BfF2A8307A4256c8797A3497"
```

## GET /v2/detailed-vaults/{network}/{vaultAddress}/apy

Returns only the APY breakdown for a single vault — useful when you need yield data without fetching the full vault object.

```bash
GET https://api.vaults.fyi/v2/detailed-vaults/{network}/{vaultAddress}/apy
```

### Path parameters

<ParamField path="network" type="string" required>Network name or CAIP-2.</ParamField>
<ParamField path="vaultAddress" type="string" required>Vault contract address.</ParamField>

### Response

```json
{
  "1day":  { "base": 0.0481, "reward": 0.0062, "total": 0.0543 },
  "7day":  { "base": 0.0490, "reward": 0.0058, "total": 0.0548 },
  "30day": { "base": 0.0475, "reward": 0.0055, "total": 0.0530 },
  "1hour": { "base": 0.0502, "reward": 0.0060, "total": 0.0562 }
}
```

<Note>All APY values are raw decimals. Multiply by 100 to convert to percentage.</Note>
